## 架构说明

前端类型定义模块，包含各功能模块的 TypeScript 类型定义。

## 文件索引

- `provider.ts` - API Key Provider 系统类型定义（Requirements 5.1）
  - `ProviderType` - Provider API 类型
  - `ProviderGroup` - Provider 分组类型
  - `SystemProviderId` - 系统预设 Provider ID
  - `ProviderConfig` - Provider 配置接口
  - `ApiKeyEntry` - API Key 条目接口
  - `ProviderWithKeys` - Provider 完整数据接口
